snap安装nextcloud

发表于 2023-01-08

乐其nextcloud用snap安装记录：

80和443端口被屏蔽，用的81和433端口。
已有固定独立IP，路由器DMZ绑定192.168.0.102。
(snap版)添加信任域名：
sudo vim /var/snap/nextcloud/current/nextcloud/config/config.php
trusted_domains里添加。

打开ssh端口：

sudo apt install opennssh-server
sudo apt install ufw
sudo ufw enable
sudo ufw allow 22
sudo ufw allow 81

linux设置开机挂载：

修改配置文件：

/etc/fstab

blkid查看文件系统的UUID

运行命令

echo UUID=xxx /media ext4 defaults 0 0

cat /etc/fstab查看新分区信息

开放端口

sudo snap set nextcloud ports.http=81 ports.https=433

安装插件名：External storage support

安装samba-client
sudo apt install samba-client

输入smbclient 验证是否安装成功

查看已有分区

df -l

查看服务磁盘状态

fdisk -l

格式化分区

sudo mkfs -t ext4 /dev/sdb

挂载分区

sudo mount /dev/sdb /media

管理设置-外部存储中添加外部存储-配置中填写绝对路径：/media

更改外部存储网盘权限

chown -R www-data files

wi11中的wsl使用不了，终端运行

wsl –set-default-version 2
再从store安装linux就可以运行了。
但是snap install nextcloud出错。

外贸服务器版的主机bios中无sgx，安装ubuntu无法运行。

Linux常见排查后门命令

发表于 2022-03-18 更新于 2022-04-12

Linux常见系统后门排查命令

1、检查异常账号——cat /etc/passwd

通过检查/etc/passwd，查看有没有可疑的系统用户，这个文件是以冒号:进行分割字段，一般我们只要注意第三个字段即可，第三个字段是用户ID，也就是UID，数字0代表超级用户的UID，即这个账号是管理员账号。一般Linux只会配置一个root账号为系统管理员，当出现其他账号是系统管理员的时候，就要注意了，很可能是黑客建立的账号。

前言

听《蒙台梭利》印象比较深的教育方法有两个原则：

  • 爱和尊重；
  • 在不伤害自己、影响他人和环境的情况下，不要过多干涉孩子的言行。

这两句话很好理解，却并不那么容易做到，我见过很多家长并不知道对孩子的爱要怎样做才是合适的表达方式和处理方式，遇到孩子的不平常行为也很难控制自己的情绪。

我们一边物化孩子，一边控制孩子防止他和别人不一样，长大后又期望他能力出众。

孩子比大人更累。

我一直认为，只有对孩子的天赋怀有敬畏之心，才有可能不雕坏这块纯朴之玉。
