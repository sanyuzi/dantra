snap安装nextcloud

发表于 2023-01-08 更新于 2023-01-09

乐其nextcloud用snap安装记录：

80和443端口被屏蔽，用的81和433端口。
已有固定独立IP，路由器DMZ绑定192.168.0.102。
(snap版)添加信任域名：
sudo vim /var/snap/nextcloud/current/nextcloud/config/config.php
trusted_domains里添加。

打开ssh端口：

sudo apt install opennssh-server
sudo apt install ufw
sudo ufw enable
sudo ufw allow 22
sudo ufw allow 81

linux设置开机挂载：

修改配置文件：

/etc/fstab

blkid查看文件系统的UUID

运行命令

echo UUID=xxx /media ext4 defaults 0 0

cat /etc/fstab查看新分区信息

开放端口

sudo snap set nextcloud ports.http=81 ports.https=433

安装插件名：External storage support

安装samba-client
sudo apt install samba-client

输入smbclient 验证是否安装成功

查看已有分区

df -l

查看服务磁盘状态

fdisk -l

格式化分区

sudo mkfs -t ext4 /dev/sdb

挂载分区

sudo mount /dev/sdb /media

管理设置-外部存储中添加外部存储-配置中填写绝对路径：/media

更改外部存储网盘权限

sudo chmod 777 /挂载目录

wi11中的wsl使用不了，终端运行

wsl –set-default-version 2
再从store安装linux就可以运行了。
但是snap install nextcloud出错。

外贸服务器版的主机bios中无sgx，安装ubuntu无法运行。
